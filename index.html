Capabilities

Everything a structural biologist needs

📄
Open-Access RAG
Retrieves full-text papers via Unpaywall. Falls back to library auth cookies (Zotero-style) in production, or Sci-Hub in dev mode.
Unpaywall · Sci-Hub · Library Auth
🤖
Cost-Free LLM
Routed through OpenRouter in dev (free-tier models) and JIT-hosted on your HPC via vLLM or Ollama in production. Zero API billing.
OpenRouter (dev) HPC-hosted (prod)
⚙️
HPC Command Tool
Submit short GROMACS jobs directly from the sidebar. SSH-routed to your cluster, with an allowlist, 5-min timeout, and local fallback for workstations.
gmx mdrun · SLURM · PBS
🧬
Structure Viewer
Interactive 3Dmol.js viewer for any PDB ID. Hotspot detection via Cα contact density, DynaMut ΔΔG predictions, and domain maps from UniProt.
3Dmol · PDB · UniProt
🔍
Sequence → Structure
Paste a FASTA sequence and PEAT resolves the best matching PDB entry via RCSB sequence search, then proceeds with full analysis automatically.
BLAST · RCSB · SIFTS
⚗️
Catalytic Site Annotations
M-CSA active site data merged with UniProt functional comments, summarised by the LLM into Structure / Function / Sequence bullets.
M-CSA · UniProt

Quick Start

Get running in 3 steps

# 1. Clone the repo
git clone https://github.com/KhareLab/PEAT.git
cd PEAT
# 2. Install dependencies
pip install -r requirements.txt
# 3. Configure secrets (.env or Streamlit secrets)
OPENROUTER_API_KEY=your_key_here
UNPAYWALL_EMAIL=your@email.edu
# Optional — dev RAG fallback
SCIHUB_ENABLED=true
# Optional — HPC execution
HPC_HOST=cluster.university.edu
HPC_USER=yournetid
# 4. Launch
streamlit run app.py

Get a free OpenRouter API key at openrouter.ai. Free-tier models (Mistral-7B, Llama-3) are sufficient for development. Point LLM_BASE_URL at your HPC vLLM endpoint for production.
